## What you can export

- PilotScore histories and deltas
- Blueprint submissions, tiers, and approvals
- Suitability decisions and follower acknowledgements
- Execution-confirm mismatches with remediation steps
- Audit logs for webhooks, overrides, and admin changes

## Formats

- JSON via API endpoints
- CSV for bulk reviews
- Signed PDFs for regulator-facing bundles (optional)

## Governance

- Restrict export permissions to Compliance and Admin roles.
- Store exports in audited storage with retention aligned to your policy.
- Regenerate exports before exams to ensure the latest evidence and signatures.
