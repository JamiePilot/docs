## Base URLs

- Sandbox: `https://api.sandbox.pilot.markets/v1`
- Production: `https://api.pilot.markets/v1`

REST APIs with predictable URLs, standard HTTP verbs, and JSON payloads.

## First call

```bash
curl "https://api.sandbox.pilot.markets/v1/navigators" \
  -H "Authorization: Bearer $PILOT_SANDBOX_KEY"
```

See [Authentication](/api-reference/authentication) for key scopes and [Errors](/api-reference/errors) for response formats.
